# 🚗 Car Showcase Website

A modern, responsive **Car Showcase Website** built with React. Users can browse cars, view detailed information, and explore a clean, interactive UI.

---

## 🔗 Live Demo
👉 https://car-showcase-liard-beta.vercel.app/

---

## 📌 Features

### 🏠 Hero Section
- An eye-catching heading
- Background banner image
- Call-to-action button

### 🚘 Car Listing
- Displays multiple cars 

### 🔍 Car Details View
- Displays:
  - Full-size image
  - Car features

### 📩 Contact Section
- Functional form:
  - Name
  - Email
  - Message

### ⭐ Bonus Features
- 🔎 Search cars by name
- 🎯 Filter by brand or price
- ✨ Smooth animations and hover effects

---

## 🛠️ Tech Stack

- ⚛️ Next.js
- 🎨 Tailwind CSS

### 📦 Libraries Used
- **react-icons** – Icon library for UI
- **lucide-react** – Modern icon set
- **react-hot-toast** – Toast notifications
- **swiper** – Carousel/slider functionality
